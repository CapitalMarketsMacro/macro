Hard rules

• Green/red are directional only — price, P&L, bid/ask. Never for success/error. Use neutral status pills.
• Brand cerulean is chrome only — CTAs, tabs, focus, live dot. Never in data.
• Every number is mono + tabular. Prices align on the decimal.
• Borders express hierarchy, not shadows. 1px hairlines. No gradients, no blur, no glassmorphism.
• No emoji. Only ▲▼ (direction) and ● (status dots) as glyphs.
• Tick flash = 600ms linear background fade. No bounces, no shimmer.
• Copy is terse, verb-first, Sentence case on actions, ALL CAPS on labels. Never Title Case.
