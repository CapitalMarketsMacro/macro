Install — AG Grid v33

// ESM
import { AgGridReact } from 'ag-grid-react';
import './themes/ag-grid/macro-overrides.css';     // tick-flash + density
import { macroTheme } from './themes/ag-grid/macro-theme.js';

<AgGridReact
  theme={macroTheme}
  rowData={rows}
  columnDefs={[
    { field: 'bid', cellClass: 'mkt-num mkt-up' },
    { field: 'ask', cellClass: 'mkt-num mkt-down' },
  ]}
  enableCellChangeFlash />

</DataTable>

The non-negotiables

  • Directional colors are sacred. .mkt-up / .mkt-down = green/red only for bid/ask, price change, P&L. Never for success toasts or error pills.
  • Non-directional status lives on Prime's p-tag-* / p-message-* severities (info blue, warn yellow, error red-on-red, success green) — we retint these to Macro but keep the semantics.
  • Brand is cerulean. CTAs, focus rings, active tabs, live-price dot. Never in data.
  • Numbers are mono + tabular. Every cell that shows price, size, rate, or count gets .mkt-num (right-aligned, tabular-nums).
  • Density is a data attribute. data-density="tight|cozy" on any scope — body, panel, or the grid wrapper. Default (none) = dense trading-desk defaults.
  • Tick-flash is 600 ms linear. motion-flash token, no bounces, no easing. Change the token if you must — don't rewrite per-component.
  • OpenFin hooks activate with <html class="openfin">: title bars become drag regions, Prime dialog headers drag, buttons and inputs stay clickable.
